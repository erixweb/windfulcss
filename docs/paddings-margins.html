Paddings & Margins

Para ponerle paddings o margenes a un elemento, se utiliza la abreviatura p- (padding) & m- (margin). Es decir, si queremos darle un padding de 8px a un elemento, pondremos p-8.

Posibilidades para paddings

p-auto
p-0
p-8
p-12
p-16
p-32

Posibilidades para margins

Para los margenes puedes usar mt (margin-top), mb (margin-bottom), ml (margin-left), mr (margin-right).

m-auto
m[t, b, l, r]-0
m[t, b, l, r]-10
m[t, b, l, r]-20
m[t, b, l, r]-40
